Tress Tracker

Tress Tracker

Date:  March 2017

Category:  Front End Development

Client:  NSS Front End Capstone

View Repo Live Demo

TressTracker is a SPA geared toward women experiencing hair loss who wear wigs. A database of 600+ medical grade wigs was created on Firebase. Users interact with an AngularJS (1.7.x) front end that allows them to create accounts and query, save, or make notes on wigs they own. Materialize and AngularMaterial were utilized for styling.

Doublespeak

Doublespeak

Date:  June 2017

Category:  Back End Development

Client:  NSS Final Demo

View Repo Live Demo

Doublespeak is a project focused on natural language processing with Python, specifically sentiment analysis of various presidential candidates. Doublespeak takes campaign speeches from presidential candidates, breaks them into individual sentences, and then gauges the 'simple sentiment'(positive/negative) of each candidate's campaign with the created module. The IBM Watson Tone Analysis API is then utilized to expand the general sentiment into personality and emotional tone analysis. This project was created with Python, Django, NLTK, Scikit Learn, IBM Watson, and AngularJS.
